Base Score = 0% (Clamped to Max 100%, Min 0%)

1. MACD (25%): 
   IF MACD Line > Signal Line => +25

2. EMA 200 (20%): 
   IF Current Price > 200-day EMA => +20

3. Volume Ratio (15%):
   IF Price Closed Higher AND Volume > 1.5x 20-day Avg => +15
   IF Price Closed Lower AND Volume > 1.5x 20-day Avg => -15
   IF Volume < 0.8x 20-day Avg => -5

4. Bollinger Bands (15%):
   IF Price < Lower Band => +15 (Oversold bounce)
   IF Price between Lower & Upper => +7
   IF Price > Upper Band => +0 (Overbought)

5. RSI (10%):
   IF RSI < 30 => +10 (Strong buy)
   IF RSI between 30 and 70 => +5
   IF RSI > 70 => +0

6. ADX (10%):
   IF ADX > 25 AND Price > EMA 200 => +10 (Strong Uptrend)
   IF ADX < 25 => +5 (Sideways)

7. PSAR (5%):
   IF Price > Parabolic SAR dots => +5
